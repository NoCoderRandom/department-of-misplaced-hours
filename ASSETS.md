## Generated Visual Assets

All background assets were generated specifically for this project with the built-in Codex image generation tool on 2026-06-18. The original PNG exports are preserved under ignored release backups, source WebP files live in `public/assets/images/`, and the release ZIP ships the optimized copies under `dist/assets/images/`. No trademarks, copyrighted characters, readable brand marks, or third-party source images were used as inputs. File hashes and source PNG hashes are tracked in `docs/ASSET_PROVENANCE.md`.

## Procedural Audio

Ambient room beds, the future-phone click clue, and ending tone gestures are generated at runtime by `src/audio/AudioDirector.ts` using the browser Web Audio API.

| Asset | Author / Tool | Source | License / Use | Modifications | Used In |
| --- | --- | --- | --- | --- | --- |
| Room ambience loops | Custom procedural synthesis | `src/audio/AudioDirector.ts` | Original project code | Oscillators and noise buffers mixed per room | All rooms |
| Future phone clue | Custom procedural synthesis | `src/audio/AudioDirector.ts` | Original project code | Click groups of 7, 3, and 1 | Vending-machine code clue |
| Ending audio moments | Custom procedural synthesis | `src/audio/AudioDirector.ts` | Original project code | Different tone sequences per ending | Ending screens |

## Researched But Not Used

I also researched Freesound, OpenGameArt, and Pixabay for longer ambience/music. For this ship pass, I kept long ambience procedural and imported only compact CC0 Kenney SFX to avoid large files and attribution ambiguity.
