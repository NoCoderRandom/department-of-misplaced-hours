The Department of Misplaced Hours is a project-specific game build. Source code, writing, generated visuals, and game design assets are not granted for reuse unless the project owner provides separate written permission.

The production build bundles Phaser 3 and Phaser-bundled runtime components. See `THIRD_PARTY_NOTICES.md` for the required Phaser, Earcut, Simplify.js, Matter.js, EventEmitter3, and Vite runtime notices.

Third-party audio files from Kenney CC0 packs are listed in `ASSETS.md`. In source they live under `public/assets/audio/ui/`; in the release build they ship under `dist/assets/audio/ui/` with Kenney license files beside the OGG files.
